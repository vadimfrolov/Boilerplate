Boilerplate for pet projects

I've made this boilerplate from scratch to use it for new projects

You can click on button "Use this template" to start new project with this template

### Installation

```shell
npm install
```

### Run the app

```shell
npm start
```

### Create build

```shell
npm run build
```

### Create and run container

```shell
docker compose up
```
